The iteration for each node gives first its value, and then the iterations of the left and right children.<br>
For node 11 that is `[11, 5, 1, 12, 3, 6, 7, 9, 8, 2]`.

<pre>
             11
         /        \
       5            7
     /   \        /   \
    1     6      9     8
  /   \               /
 12    3             2
</pre>
